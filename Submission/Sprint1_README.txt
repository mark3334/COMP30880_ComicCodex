command line invocation:
to run the file do java -jar Sprint1executable.jar
type in your prompt and type 'exit' to quit the program.

code structure:
The OpenAIClient class is the main class which is responsible for
sending and receiving JSONobjects and creating and parsing them with
strings. The client gets the api_key and other parameters from the config.txt file
using the ConfigurationFile Class which has methods for getting value by key
and creating a hashmap from the Config.txt file
